# Unveiling the Android App Market

## Oasis Infobyte Data Analytics Internship (OIBSIP)

### Task Level
**Level 2**

### Objective

The objective of this project is to perform Exploratory Data Analysis (EDA) on the Google Play Store dataset to uncover valuable insights into the Android application market. The analysis focuses on app categories, user ratings, installs, pricing strategies, and user engagement to better understand market trends.

---

## Dataset

The dataset contains information about Android applications available on the Google Play Store.

### Features

- App
- Category
- Rating
- Reviews
- Size
- Installs
- Type
- Price
- Content Rating
- Genres
- Last Updated
- Current Ver
- Android Ver

## Data Preprocessing

The following preprocessing steps were performed:

- Removed duplicate records.
- Removed the corrupted record from the dataset.
- Handled missing values using appropriate techniques.
- Converted Reviews, Installs, Price, and Size into numerical formats.
- Converted the Last Updated column into datetime format.
- Prepared the dataset for analysis.

## Key Findings

- Most applications have ratings above **4.0**, indicating generally positive user feedback.
- Communication and Social categories record the highest average installs.
- Free applications dominate the Google Play Store.
- Most paid applications are priced below **$50**.
- Reviews and installs show a strong positive relationship.
- User ratings remain consistently high across different content rating categories.

---

## Business Recommendations

- Focus on high-demand categories such as Communication and Social.
- Maintain high application quality to achieve better ratings.
- Encourage user engagement through reviews and feedback.
- Consider free or freemium monetization strategies.
- Keep application pricing competitive.
- Regularly update applications to improve user satisfaction.

---

## Conclusion

This project demonstrates how Exploratory Data Analysis can be used to understand market trends, user preferences, and application performance in the Google Play Store ecosystem. The insights obtained can support developers and businesses in making data-driven decisions related to application development, pricing, and user engagement.
